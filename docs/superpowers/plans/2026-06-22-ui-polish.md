# UI Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Apply 5 surgical UI fixes — palette warmth, hero simplification, glass/grain removal, animation easing, and dark-section accessibility — without touching page routes, data files, or booking logic.

**Architecture:** All changes are in-place edits to existing component and config files. No new files created. Each task is self-contained and can be visually verified in isolation by running the dev server (`npm run dev`).

**Tech Stack:** Next.js 14 App Router · TypeScript · Tailwind CSS · Framer Motion

import { motion } from 'framer-motion'
import { HEALOW_URL } from '@/lib/constants'

interface BookingCTAProps {

  className?: string

  return (

      href={HEALOW_URL}
